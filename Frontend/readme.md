# Frontend
## Tools
* **React.js** for Component Architecture.
* **axios** for HTTP requests.
## Installation
1. Rename src/login.js to src/Login.js (the capital "L" is important and gets lost in translation for some reason)
2. `npm install`
3. `npm start`
## Component Architecture Hierarchy
* Main
  * Entry
    * Home
    * Login
  * Hub
    * Home
    * Collection
    * Gallery
    * Log Out
